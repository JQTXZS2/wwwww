# AI 工具使用声明

本项目开发过程中合理使用了以下 AI 辅助工具：

- 工具名称：OpenAI Codex；
- 相关大模型：GPT-5；
- 使用场景：阅读和梳理 Asterinas、dm-crypt、dm-verity 相关代码；辅助分析
  BIO 数据路径与构建错误；生成或修改部分 Rust、Shell 测试和构建脚本；
  协助整理 README、设计说明、测试状态及参赛文档表述；辅助执行测试、
  完整内核构建和 QEMU 验证并解读日志。
- 工具名称：通义千问（Qwen）；
- 使用场景：技术文档润色和中英文专业术语校对。
- 工具名称：豆包；
- 使用场景：答辩表述优化与常见问题整理。

AI 工具仅作为开发辅助。项目成员对采用的代码和文档进行了人工审阅，
并通过单元测试、Linux dm-crypt 逐字节对照、Asterinas 内核构建及 QEMU
运行验证其正确性。最终的技术方案、代码取舍、提交和参赛责任均由参赛
团队承担。

相关 AI 交互记录保留在项目对应的 Codex 开发线程中；使用 AI 参与的提交
在 Git 提交说明中注明工具名称、模型和使用场景，便于后续审计与披露。
